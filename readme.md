### 技术选型

- web层 
    - Servlet 前端控制器
    - html 视图
    - Filter 过滤器
    - BeanUtils 数据封装
    - Jackson json序列化工具
- Service层
    - JavaMail 邮件客户端
    - redis nosql内存数据库
    - Jedis java的redis客户端
    
- Dao层
    - mysql 数据库
    - druid 数据库连接池
    - JdbcTemplate jdbc工具
